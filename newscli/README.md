# gRPC Client (cli) example

## Build & Run client

Project require Go installed.  

1. install newscli : 
```SH
go install github.com/fb64/grpc-fullstack-demo/newscli@latest
```

2. Start server (see [backend project](../backend/))

3. run newscli :
```
newscli -h
```


## Libraries used

* [urfave cli](https://github.com/urfave/cl)
